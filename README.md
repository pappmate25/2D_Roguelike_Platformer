## About
Sewer Dungeon is a 2D roguelike platformer built solo in Unity for a university competition, where it placed 2nd among 10+ teams.
Players fight through hand-built levels, collecting currency to invest in build-defining stat upgrades between levels.
The design goal was to build a short but challenging replayable prototype.

## Design Highlights
- **Three-life run structure** — preserves run tension without full restart punishment after a single mistake;
  losing all three lives sends the player back to level 1.
- **Currency drops + between-level upgrades** — gives players a 
  meta-progression loop that respects the short session length.
- **Crit chance vs. crit multiplier as separate stats** — creates a 
  meaningful build trade-off (consistency vs. burst), echoing 
  Path of Exile and Diablo build identity.

## Controls
- **Left:** A
- **Right:** D
- **Jump:** W/Space
- **Dodge:** LShift
- **Close shop:** P
- **View Stats:** E
